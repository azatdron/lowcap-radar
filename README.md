## Идея
Это не копия CoinMarketCap. Цель — искать перспективные lowcap-монеты до сильного роста.

Пользователь выбирает только:
- сектор / narrative
- сети
- риск
- цену монеты

AI-карточка показывает:
- капитализацию
- рост объёма
- рост цены
- ликвидность
- возраст проекта
- соцсети
- комьюнити
- фонды
- экосистему
- narrative / сектор
- риски
- почему AI выбрал монету
